    atributos y valores

    Los atributos y valores de HTML son en especial palabras especiales que se utilizan dentro de una etiqueta ya se para apertura y también para cerrar y poder controlar el movimiento y comportamiento de los elementos sus valores dependen de los modificadores existen varios atributos requeridos necesitados por un tipo de elemento particular Atributos opcionales utilizados para poder modificar la funcionalidad ya sea por defecto Atributos estándares estos soportan por mucho los tipos de elementos Atributos de evento estos se utilizan para causar los tipos de elementos ya sean guiones y ser guados bajo circunstancias concretas
